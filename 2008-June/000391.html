Dear all,

A current gcc is causing problems when building MRS. In particular it is 
version:

 > gcc --version
gcc (GCC) 4.2.3 (Ubuntu 4.2.3-2ubuntu7)

The symptoms are weird crashes when doing a query that contains an OR 
construct.

The solution is to build MRS with -O2 instead of -O3, you'll have to 
edit the mrs/plugin/Makefile for this. Search for OPT in this file.

-maarten
