Uli schreef:

> I have been trying to change the data dir for the webservices, but it 
> seems to default back to the original data directory MRS_DATA_DIR = 
> /usr/local/data.  I have tried the following:
> 
> 1.
> ./configure.pl --data-dir /mypath/mydatadir
> make clean
> make
> make install

A make install does not overwrite any existing mrsws-config.xml file 
that may be in /usr/local/etc/. You will have to edit that file 
manually. Whenever a new MRS is installed and a new config file is 
written it is place in /usr/local/etc/mrsws-config.xml.dist
You can this .dist file as a reference to see if anything changed 
between the releases.

I'm sorry, this should have been documented of course.
