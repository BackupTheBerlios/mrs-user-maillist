OK, it took some time, but I believe 3.1 is a release. I've made a 
branch in the subversion tree at berlios. One day I will even create a 
down loadable tar ball for public consumption.

And now I will start working on MRS 4. So please be careful when you 
check out source code directly from berlios, the trunk will be quite 
unstable from now on until I post a notification here.

First things I will implement are:

- better support for phrase search and a way to look for words that are 
'near' to each other in the original text
- better UTF-8 support
- direct XML import without the need to write parsers, simply specify a 
mapping from XPath to index.
- protected MRS files or some mechanism to inhibit access to private data.
- and perhaps query relaxation using thesauri?
