Hi,

I've checked in a version of MRS that compiles again, version 986. You 
can get the source code from the svn repository at the usual location. 
Be warned though that I haven't fully tested this version yet.

This version uses libzeep, which is a spin off project of MRS. I've 
created a separate Berlios project for it and you can download the 
source code for libzeep from:

http://developer.berlios.de/project/showfiles.php?group_id=11605

You need to install this package first before you can build MRS.

I noticed something else, modern Linux systems are distributed with the 
latest GNU C++ compiler which is version 4.4 at the moment. This version 
is not binary compatible with C++ libraries compiled with version 4.2 of 
GCC. Be very careful if you have more than one compiler on your computer.

-maarten
