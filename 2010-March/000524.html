Thanks to the Maarten's help and thanks to the web-services ability of the
MRS, I was able to put together a Perl module that can be used to access MRS
services (queries, blast and clustal) pragmatically. The module is called
MRS::Client and its first version was released yesterday to the CPAN Perl
repository, available for download from there:
http://search.cpan.org/~tulsoft/MRS-Client-0.52/. Documentation can be seen
at http://search.cpan.org/~tulsoft/MRS-Client-0.52/lib/MRS/Client.pod.

The part of the module distribution are also command-line programs (that use
the MRS::Client module) and that can be used without Perl programming
knowledge (their names are mrsclient, mrsblast and mrsclustal).

I would appreciate any feedback on the module and I very welcome suggestions
and bug reports.
