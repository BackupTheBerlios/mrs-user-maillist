	Dear Felipe,

You can download the wrappers4EMBOSS suite from http://wemboss.sourceforge.net/.
It has a module MRS that allows to use MRS as databank access tool for EMBOSS 
(and also a scripting tool and an EMBOSS program that looks like GCG lookup). 
The README file explains in detail how to proceed.

If you don't feel like doing a full installation and just want the databank 
access you can just take the script 
.../wrappers4EMBOSS-2.3.0/BENwrapper4MRS/mrsseqget.pl
and replace the word PERL in the first line by /usr/bin/perl or whatever it is.
Then you must in emboss.default or in  /home/jdoe/.embossrc put items :

DB uniprot [ type: P methodquery: app format: swiss
              comment: UniProt
              app: '/usr/local/bin/mrsseqget.pl uniprot %s'
]

DB genbank [ type: N methodquery: app format: genbank
              comment: 'GenBank last release'
              app: '/usr/local/bin/mrsseqget.pl genbank_release %s gi'
]

Note that EMBOSS has a built-in method for MRS, but that is for retrieving just 
one sequence at a time from a WWW server, not really what you need. To use it 
you must put in emboss.default or in  /home/jdoe/.embossrc items like

DB uniprot[ type: P  comment: UniProt
     methodentry: mrs3 dbalias: uniprot format: swiss
     url: 'http://mrs.cmbi.ru.nl/mrs-web/plain.do'
]
