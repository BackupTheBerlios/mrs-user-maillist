dbEST is an EST subset of Genbank. If you use MRS, you could retrieve 
sequences in FASTA format by asking Genbank directly. Use your sequence 
search terms and make sure to include the index keyword (for Genbank). 
For example:

keywords:EST Myxococcus xanthus


This should allow you to return the sequence(s) in question in FASTA format.
