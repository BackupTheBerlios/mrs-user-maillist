You can now download a distribution for MRS 6 from (same code as used for http://mrs.cmbi.ru.nl):



ftp://ftp.cmbi.ru.nl/pub/software/mrs/m6-6.0.0b1.tbz



This tar contains a read-me describing very briefly how to build and start using mrs.



The only requirements left on Linux are:



gcc version 4.6 or equivalent

boost libraries 1.48

libperl-dev



All the other dependencies have been removed.



Configuration is now done mainly through a web interface.



I will now spend the rest of the year on writing documentation. Unless bugs show up.
