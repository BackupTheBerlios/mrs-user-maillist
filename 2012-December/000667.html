A quick update. If you installed the first beta, you will have to review 
the new configuration file carefully. A new file is written to 
/usr/local/etc/mrs/m6-config.xml.dist. Or simply remove your current 
config file and run make install. I changed the location of the raw 
files to make them compatible with a mrs 5 installation.

I also rewrote the FTP client since the one using ASIO was hanging when 
behind a firewall. (No, not a simple passive problem).

To get you up and running quickly, I copied the output of an install and 
configure session. When completed I was able to access a running MRS at 
http://192.168.74.133:18090/

-maarten
