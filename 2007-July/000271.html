	Dear Maarten, dear all,

Rebase is a very useful database with information about restriction 
enzymes. At the BEN site we always had it available in our SRS server 
and, since we did not want to lose it, I did the effort to integrate in 
our MRS server. For those who are interested I send you my version of 
the parsers.
A question was which format to use (R. Roberts provides versions of 
Rebase in a great variety of formats). I chose to use the "bairoch" 
file, which is also used by SRS. It is easy to parse and is rich in 
information. A minor trouble is that the "bairoch" file contains two 
entries with same ID for enzymes like BcgI that make 4 cuts, but this is 
not really toxic.
There is a second table with the commercial data. You will note that at
http://tanga.ulb.ac.be:8080/mrs-3/ it is e.g. possible to search in 
Rebase for target:cccggg and in an entry to follow a hyperlink that 
gives you the list of commercial providers.

	Regards,
	Guy Bottu,
