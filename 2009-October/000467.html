Hi,

I've uploaded a MRS rpm to ftp://ftp.cmbi.ru.nl/pub/software/mrs The source rpm is there as well.

Took some time to create this, never created an rpm before. The rpm's have a few dependencies you'll have to solve before installation will succeed. One dependency I could not add though: XML::XSLT or XML::LibXSLT and so I commented out the 'use XML::XSLT' in the parser scripts that use this module. You'll have to uncomment these lines yourself and then install XML::XSLT and XML::LibXSLT by hand using cpan if you want to index interpro, medline or gene.

Looking forward to feedback.

BTW, this is MRS 4.1.1 which is the new stable version. I'll create a tag for it in the repository and make a normal source distribution as well. It is a very stable version of MRS, at least it is for us here at the CMBI.

Best regards,

-maarten
