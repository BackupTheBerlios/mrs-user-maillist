	Dear colleagues,

I just noticed that in the most recent release of Pfam they started to 
use version numbers instead of accession number to reference the UniProt 
entries. A result is that the hyperlinks do not work anymore. But it is 
easy to fix, e.g. by replacing in Format_pfam.pm
match	=> qr|^(#=GS .+AC )(\S+)|,
by
match   => qr|^(#=GS .+AC )([A-Z0-9]+)|,

There is a similar problem with hyperlinks from Pfam-B to Pfam-A, which 
can be solved by replacing
match   => qr|^(#=GF DR\s+PFAMA;\s)(\S+)(?=;)|,
by
match   => qr|^(#=GF DR\s+PFAMA;\s)(PF\d+)|,

You can look as example to pfamb ac:PB020936
