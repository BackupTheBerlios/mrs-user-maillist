I'm afraid this problem is not related to the data but to the way it  
is extracted in the MRS scripts. The data itself is in two tar files.  
These files are then extracted using the O flag of tar and output is  
redirected to a new file. This assumes that the tar utility only  
outputs the text data from the tar, perhaps your tar inserts binary  
data?

If this tar trick is not reliable I'm afraid I will have to change  
the script to extract the tar in the normal way and then revise the  
parser script to read all the separate files.
