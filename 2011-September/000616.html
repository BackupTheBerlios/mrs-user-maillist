Op 11-9-2011 9:16, Martin Senger schreef:

> Before MRS5 I was calling makefile with the target 'mrs', for example:
>
> make DATABANK=aloha mrs
>
> Can I do the same (or similar) with the mrs-update?

Yes, you can, simply substitute make with mrs-update:

mrs-update DATABANK=aloha mrs

So to test this, without actually running the update, you can run:

mrs-update -n DATABANK=aloha mrs

The only thing that changed is that the former makefile is now called 
mrs-update and I made it executable.
