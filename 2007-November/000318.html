Dear all,

I've just checked in a new version of MRS which should install much easier.

On my new desktop (an OpenSUSE 10.3 system, x86_64) I have to type this:

$ perl configure.pl --cc='g++-4.1' --boost_regex=boost_regex
$ make
$ make install

and then

$ cd /usr/local/share/mrs/update_scripts
$ make enzyme

I hope it works similar for you. Please let me know if there are any
remaining problems.

-maarten
