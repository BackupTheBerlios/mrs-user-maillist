> I am still not quite clear what I need to do to get the web services to
> work though

You need to install the mrs.war web application in a Tomcat or other
servlet container. If this tomcat is running on another computer you
have to configure it to use your mrsws server. You can do this by
editing the MRS environment with the Tomcat Admin tool that comes with
tomcat.
