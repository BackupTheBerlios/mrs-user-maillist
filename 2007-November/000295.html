Uli Horn schreef:

> How do I set the MRS software up as a web service?

Suppose you use the source code from Berlios, you simply install the
entire package. Like this:

svn co https://svn.berlios.de/svnroot/repos/mrs/trunk mrs
cd mrs
perl configure.pl
make

Well, in theory, that's how it should work. This should install the
applications mrsws, mrsws-blast and mrsws-clustal.

The applications are build in the mrs/ws/* directories. In each
directory you can find a Sources directory and there the wsdl file for
each service is located.

The ws/init.d directory contains init.d scripts for RHEL to
automatically start the web services. You have to install those by hand.

Maybe this is enough information, but don't hesitate to ask for more.
