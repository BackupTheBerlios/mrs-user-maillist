	Dear users of MRS,

You have probably heard about EMBOSS, a software package for sequence analysis, 
developed by a team headed by Peter Rice and Alan Bleasby at the European 
Bioinformatics Institute. wrappers4EMBOSS is a suite that allows to integrate 
3rd party software like BLAST and MUSCLE under EMBOSS. It has very recently been 
enriched with a new module "MRS". The "MRS" module uses Perl scripts to access a 
local installation of MRS version 3. I contains the following :
- a script that allows to use MRS as sequence database access for EMBOSS, so
   that you can let any EMBOSS program operate on a "public" sequence by typing
   simply e.g. swissprot:papa1_carpa
- a program mrsindexsearch that allows to search in sequence databanks by
   providing keywords for the different fields and that returns the result as
   either a multiple sequence file in fastA format or as an EMBOSS List File (a
   list of names that can directly be given as input to an EMBOSS program). The
   program can be run interactively in a UNIX terminal or it can be integrated
   under a GUI like wEMBOSS, where it presents a page with multiple boxes.
- a scripting tool mrsget (which actually has nothing to do with EMBOSS) that
   allows to type in a query in MRS query language and retrieve the result in any
   of a series of formats.
You can download it from http://www.wemboss.org/. The included README and 
0README1 files explain how to proceed.
