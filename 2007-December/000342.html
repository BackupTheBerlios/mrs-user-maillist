This means that the web application is working properly, but there is  
no mrsws running.

So first you should make sure that mrsws is built properly, then you  
need to launch this application by hand or by using the scripts in  
mrs/ws/init.d (install those in /etc/init.d and use the RedHat  
service command to start mrsws, e.g.: 'sudo service mrsws start')

Then the web app should be able to connect to the server.

The mrsws server creates a log file named /var/log/mrsws-search.log  
and mrsws-blast and mrsws-clustal have a log file too.

I'm not so sure about the java/tomcat error you're seeing. And I'm  
not sure it is relevant here.
