Uli schreef:

> My "production" server is running the 518 svn version.  I would like to 
> upgrade to the latest  version.  Can I just stop the services and 
> download and compile the latest version or do I need to delete files in 
> the /usr/local/bin  ../etc ../share/mrs/... etc directories and the 
> MRS.pm and MRS.so files

You can install a new version of the older, you don't have to deinstall 
anything.

But I would not upgrade to the trunk version of MRS right now. I cannot 
guarantee it works without any problem. But you can take e.g. the latest 
release version which can be found at 
https://svn.berlios.de/svnroot/repos/mrs/branches/
