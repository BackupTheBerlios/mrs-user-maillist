I've updated both libzeep and mrs.

libzeep had to be updated because amd64/gnuc decided it was time to 
change the wordsize from 32 bit to 64.

mrs was updated because there was a serious bug in the SOAP blast code, 
the wrong IDs and titles were returned for blast hits in joined databanks.

I've put the updates as source tarballs at:

libzeep: ftp://ftp.cmbi.ru.nl/pub/software/libzeep/libzeep-2.6.2.tgz
mrs:     ftp://ftp.cmbi.ru.nl/pub/software/mrs/mrs-5.1.3.tgz
