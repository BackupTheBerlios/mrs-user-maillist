Well, the error message from the linker is clear. I cannot find a  
correct libboost_regex file to use. Since you seem to have multiple  
versions of libboost_regex I'd suggest you use the 'locate' tool to  
find all directories containing libboost files. Then compare your  
list with the list in the install.pl script at line 225. If you have  
a directory not mentioned there, try to add it to the script and run  
it again. Maybe that works.

ask for an installation directory before trying to be smart. Or
