	Dear Maarten,

I noticed that in GenBank and RefSeq, as it is currently indexed, the 
"project" index contains only one key "genomeproject". This is of course 
not interesting. It is however sufficient to add in the file genbank.pm 
the lines :

      elsif ($field eq 'project' and $text =~ /GenomeProject:(\d+)/o)
                {
                       $m->IndexWord('project', $1);
                }

and we get an index with the project identifiers (which happen to be 
things like 18017)
