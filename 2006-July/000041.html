My guess is your boost is installed in a different location than  
default. Just saw this same problem with someone else.

First try to locate your boost folder. Perhaps it is in /usr/local/ 
include/boost-1_33_1/boost or something. Note the path in which the  
boost folder is located. Now edit the install.pl script, lines  
194-196 are commented out, you can uncomment them. In that case the  
install script will ask for the correct boost location that you can  
now enter.

This should work, if it doesn't you can comment out the die statement  
in lines 199 and 200. In that case you have to edit the make.config  
file yourself to add the correct path to boost.
