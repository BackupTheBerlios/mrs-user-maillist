Hi Diego,

>  I get the following error in the log file after executing "make  
> interpro".

> Can't locate object method "version" via package "interpro::parser" at
> /raid/usr/local/bin/mrs.pl line 290.

This is not really an error, the only error here is that the message  
looks like an error. The problem is, the interpro parser does not  
provide a method to determine the version of the current data. E.g.  
SwissProt has a method called 'version' in the uniprot.pm file that  
parses the reldate.txt file to find out what version the files are.  
Some parser scripts should get a version method as well, but for some  
it is simply impossible since there is no file containing a version  
string and for these I will have to create some generic file date  
based version string generator.
