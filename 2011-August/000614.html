Dear all,

Our main MRS server at the CMBI is running Ubuntu 10.04 LTS and I used 
Debian Squeeze to develop MRS. No problems there. However, I recently 
tried to run MRS on a machine with Ubuntu 11.04 and found out that 
there's something wrong, a mrs-build created on this OS was not capable 
of creating MRS files. I suspect the compiler, Ubuntu 11.04 comes with 
gcc 4.5.2, this is something I have to investigate.

So in case you run into problems with a MRS you try to build yourself, 
you can try to build MRS with e.g. gcc 4.4.3 (the one we use) or you 
should contact me. Of course, when using Ubuntu, your best option is to 
use the prepackaged version of MRS from the PPA apt repository.

-maarten
