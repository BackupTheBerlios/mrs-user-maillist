Guy Bottu schreef:

> Is there somewhere documentation about how to write a Web services client for 
> MRS version 4 ? Can it still be done in Perl SOAP::Lite or does it now involve 
> SOAP::WSDL ?

SOAP::Lite should still work, however, I've put version 4.1 online today
and now the WSDL has changed. I did this to make it more compatible with
Python (I got complaints about the use of hyphens in field names).

Anyway, you can also use XML::Compile and friends to do the webservices
work from Perl. It looks very promising, but there are problems, like
the need to specify all parameters.

If you look at the embrace webservices page at
http://prints.cs.man.ac.uk:8081/webservices
you can find the test scripts for many of the registered services. The
MRS test scripts are using XML::Compile now.

And to get your scripts working again, the main thing that changed is
that I removed all underscores and hyphens and replaced them with Java
style upper-case letters.

Hope this helps?

-maarten
