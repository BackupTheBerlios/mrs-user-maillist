> I was thinking about adding few mrs tests to emboss qa tests. I have
> noticed that above query returns its results in an xml structure that
> looks like a soap envelope, even if i select the format 'plain'. May
> this going to change in future?

You're right, the result of this REST call is a SOAP envelope. Exactly 
the same as would have been returned by the corresponding SOAP call. In 
the case of retrieving just one piece of data this may seem a bit of 
overkill, but I left it that way since other calls might return more 
complex data and then I would have to come up with a way to format that.

However, if you're only interested in plain text records, you might as 
well use the 'download' facility in MRS:

http://mrs.cmbi.ru.nl/mrs-5/download?db=sprot&id=104k_thepa

This returns the records in plain text. Maybe that's a better option for 
you?
