I've released mrs 6.0.2 today. There were some important changes:


-          The m6 application and all configuration files with an m6 in their name have been renamed to mrs. I thought this was more consistent.

-          The webservices API was updated to work with MRS::Client (as announced before).

-          Some changes were made to the installation procedure to allow easier Debian package creation.

And as that last change implies, I've submitted MRS to Debian for inclusion into the next major release. Hopefully it will be accepted, and if so, it will trickle down into Ubuntu one day as well.

Oh, and documentation still sucks. Please ask questions here to find out how things work.
