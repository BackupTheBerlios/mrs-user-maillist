Dear Heikki,

Our main MRS server is running Ubuntu 8.04. On this computer I downloaded and installed boost version 1.36 and I specified this accordingly when running the configure.pl command. Building boost takes some time, but it is needed if you want to use an OS that does not come with version 1.36 or later.

The warning messages about missing files when running make are really only warnings, I don't know of any way to suppress them and they go away after you've build MRS one time. If the make dies then there must be some real error message hidden somewhere in the output.

The (subversion) source code repository can be found at: https://svn.cmbi.ru.nl/mrs/

I stopped creating regular releases since I had the impression there were not that many users of MRS still around. I do continue developing MRS, albeit at a slow pace, since we use it ourselves quite heavily. Whenever I release something, I try to highlight the main points in this mailing list.

If you still have problems creating a binary version of MRS, let me know and I can send you my binaries.

Best regards,

-maarten hekkelman
