I hoped to finish the development of MRS 4 before summer.  
Unfortunately, I'm almost there but as usual the last 10% takes 90%  
of the time.

Anyway, you can see the new mrs in progress at http://mrs.cmbi.ru.nl/ 
mrs-web/

You won't notice that many changes for now, with an exception of the  
databank overview page where you can see databanks being updated.  
More status information will appear here soon.

You can see the parser script for a databank by clicking its name in  
the databanks info page.

And most databanks now can be searched using phrases, like in:

	"retinal degeneration slow"

or even

	de:"retinal degeneration slow"

Unfortunately, I still have to do some work to fix the indexing of  
very large databanks with this phrase search support in the index.  
I've disabled this feature for now for EMBL, genbank and dbest.

The whole process of updating has been streamlined. I believe it is  
much more simple now.

As I said, things are not finished yet, but I'm getting closer.
