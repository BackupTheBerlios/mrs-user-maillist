I've started documenting the MRS interface. You can find the first  
results in HTML at http://www.cmbi.ru.nl/mrs/doc/

The documented interface is the C++ one. However, this interface is  
processed by SWIG and the resulting Perl interface uses the same  
classes and methods. The only difference is syntactical.

I hope this documentation is useful as a first start in using MRS in  
your own scripts or applications.
