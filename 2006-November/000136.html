In that case you really have to check the permissions of the full  
path to the mrs files. If one of the directories between / (root) and  
your file is not world readable and executable you cannot access the  
file from within the mrs script.

To verify this, you can add the following to the mrs.cgi script at  
line 937 (following the line 'my $searchBox;')

print "trying to open file $mrs_data/go.cmp\n";
my $sb = stat("$mrs_data/go.cmp");
print "Could not access file go.cmp" unless defined $sb;
while ($mrs_data ne '/') {
	my $sb = stat("$mrs_data");
	print "Could not access dir $mrs_data" unless defined $sb;
	$mrs_data =~ s|[^/]+/?$||;
}

	
This should give you an indication where things go wrong.
