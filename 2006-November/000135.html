Things you can try:

- in a terminal, go to the cgi-bin directory and type:

$ perl mrs.cgi overview=1

this way you start the mrs.cgi script from the commandline and it  
should give you the HTML output, not some kind of error. If this  
doesn't work you probably have an error in the mrs.conf file in cgi-bin.

If this command works properly though, you could try to become the  
web user and then run the same command to see if permissions are all  
correct.

And if that works correctly as well, you should let me know so I can  
try to think of some more things you might check.
