If you plan on building EMBL or other large databanks you'd best buy 
multicore hardware with sufficient memory. We have 32 GB of main memory 
which is good enough. The disks we're using are MD1120's from Dell, a 
very fast and reliable 2.5" RAID cabinet that can be expanded (we just 
added a second enclosure).

As for the OS, best is to use the same as I use: Ubuntu 10.04 64-bit. 
(Lucid)

To install you could follow the instructions here:

http://mrs.cmbi.ru.nl/mrs-web/help.do?page=50

But I think it may be better to use the HEAD revision from the SVN 
repository right now. To do so, you should check out the source code 
using subversion. First install subversion (apt-get install subversion) 
and then type:

svn checkout https://svn.cmbi.ru.nl/mrs/trunk mrs

And then continue the process as indicated in the help pages. The help 
page hasn't been updated in some time, but I think they're still useful.

If you run into trouble, let me know and I'll try to help you out. I 
wouldn't be surprised if there are still some small problems left in the 
code. I haven't had time to polish it for a new release.
