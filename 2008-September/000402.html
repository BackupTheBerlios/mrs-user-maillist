> Well, for what it is worth (we are still with MRS 3), at the BEN site we have
> had several times the same problem as you and we did the following :
> in /usr/local/share/mrs/update_scripts/make.post outcomment the line
> rm -rf ${PARTS:%=$(MRSDIR)$(@F:.cmp=)-%.cmp}
> so that the intermediate partial indexes ar preserved :
> -rw------- 1 root root  10148494152 Jun 14 16:41 embl_release-1.cmp
> -rw------- 1 root root   9243439972 Jun 14 16:47 embl_release-2.cmp
> ...
> -rw------- 1 root root  10454405416 Jun 16 00:11 embl_release-32.cmp
> If just the merging step failed, it is easy to re-do it with the command
> /usr/bin/perl /usr/local/bin/mrs.pl merge -d embl_release -P 32 -s embl
