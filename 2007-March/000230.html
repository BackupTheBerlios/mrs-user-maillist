Op 19-mrt-2007, om 18:14 heeft Guy Bottu het volgende geschreven:
>>
>> I think that with these changes--along with a couple of small fixes I
>> checked in today--I've finished the transition from MRS 2 to MRS 3.
>> All the functionality that was there in MRS2 is now available in
>> MRS3. Unless I've overlooked something of course.
>
> Well, a few things are still lacking :
> - the "wildcard" check box

I removed that one on purpose. MRS is not SRS and using the wildcard  
checkbox would mean you can no longer use vector matching. And if you  
really want to append wildcards it is not that difficult to type them  
in yourself.

> - the possibility to save sequences in fastA format

That's a bug, you're right. Should have added it. I'll see if I can  
fix this.

> - the possibiliy to search for groups of word (the famous "abnormal
>   auricles")

The way this was implemented was a hack. And a hack that took way to  
many resources on the server. I will re-implement this feature very  
soon again but in a decent way. I'm afraid this means I will have to  
extend the MRS file format for this and that will take time.

> - the on-line help about the MRS query language

Yes, I really have to do something in that area.
