Dear all,

I've checked in a couple of changes into the trunk today. First of  
all I've reimplemented the blast against a query in the databank. For  
this I had to modify the web services and the webapp. I also changed  
the code in the MRS library to properly calculate the e-value for  
hits when searching against an MRS query.

I think that with these changes--along with a couple of small fixes I  
checked in today--I've finished the transition from MRS 2 to MRS 3.  
All the functionality that was there in MRS2 is now available in  
MRS3. Unless I've overlooked something of course.

I'll create a new tag on the current version called 3.0.1 and then I  
will recreate a tarball for this.

If you have suggestions for future improvements, please let me know.  
I'll start planning the new feature set for version 4 soon.
