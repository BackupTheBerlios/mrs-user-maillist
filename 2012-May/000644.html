	Dear all,

Version 2.6 of wrappers4EMBOSS has just been released (see 
http://wemboss.sourceforge.net). The module wrapper4MRS now supports MRS 
version 5. It allows to use MRS as sequence retrieval tool for EMBOSS 
and has a program indexsearch that can search in sequence databanks and 
creates List Files directly usable by EMBOSS.

A hic is that accessing a local installation of MRS by direct access to 
the Perl API will probably not work. It works on my own computer but I 
used a special version that Maarten Hekkelman sent me. However, 
accessing a remote version of MRS by Web services works fine ("remote" 
can of course also be the same computer if you put the Web services 
behind "localhost"). And since Maarten has promised that the Web 
services interface will not change, it should continue to work with
MRS 6.

	Regards,
	Guy Bottu,
