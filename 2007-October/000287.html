I recently tried MRS on an OpenSUSE 10.2 installation. I remember I 
had some problems here similar to what you describe. I believe they 
were caused by the fact that SuSE does not come with SUN's Java but 
with Kaffe. Once you've installed SUN's Java SDK and edited the 
tomcat startup scripts to use this Java then everything should work OK.

You might also want to use Tomcat 5.5 to be sure, that's the version 
I'm using myself. You can download it from http://tomcat.apache.org/

Hope this helps. Please let me know if you need more help.
