Regarding my previous email, I think I got it wrong. After looking into the
code again, it seems that there are no two SOAP servers - but just one, the
mrs-ws. And mrsweb is just a web interface using it (I was mistaken when I
noticed axis2's server configuration file in the war file). I even found the
correct service address used in a WSDL, pointing to my own mrs-ws:

 <wsdl:service name="mrsws_search">
  <wsdl:port name="mrsws_search" binding="ns:mrsws_search">
   <soap:address location="http://localhost:18081/"/>
  </wsdl:port>
 </wsdl:service>

This definitely answers my first question whether I can connect directly to
mrs-ws. Yes, I can, I believe.
