I've dropped support for MacOS X for now. It proved to be too much 
trouble trying to get MRS to run properly on that OS. And not to be 
rude, but performance on MacOS X really sucked when compared to Linux.

If you're serious about running MRS you should install a 64 bit Linux 
somewhere. And if dedicated hardware is a problem you can always fall 
back to a virtual machine, but don't expect too much performance wise in 
that case.
