> A question : can MRS extract keywords that contain blank spaces, like

> papaya' ? I have the impression that it cannot.

Yes, well, sort of.

marks. Like in this query:

<http://www.cmbi.ru.nl/mrs/cgi-bin/mrs.cgi?db=sprot%2Btrembl&query=% 
22carica+papaya%22>

However, the implementation of this search is less than optimal  
although the results should be OK.
