Dear మహేష్,

> Please help me add new indices to gene.

The steps you mentioned in your e-mail seem to be OK. Now since you have 
MRS installed, you can easily check whether the indices were actually 
created by using the mrs-test tool. Simply run:

mrs-test -d gene

And you should see a description of what's in the gene.cmp file. Your 
new indices should be there.

However, looking at the Perl code you posted, I do see one potential 
problem. You have the regular expression:

elsif ($line =~ m|^<Org-ref_taxname>(\.+)</Org-ref_taxname>|)

This expression only matches the tags with a series of one or more dots 
in between. I guess you really wanted:

elsif ($line =~ m|^<Org-ref_taxname>(.+)</Org-ref_taxname>|)

or

elsif ($line =~ m|^<Org-ref_taxname>([^<]+)</Org-ref_taxname>|)

Hope this helps.
