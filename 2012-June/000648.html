Ah! Now I understand.

There are three main methods to retrieve external data. One is rsync,
and for this you can add an environmental variable if I understand the
man page for rsync correctly.

The other tool is mrs-mirror which uses the FTP protocol. If you read
the man page for Net::FTP you'll see a reference for some kind of proxy
mode there too. Seems like the environmental variable FTP_FIREWALL
should be used here.

Finally, snarf is used for HTTP access. And the snarf man page specifies
proxy settings as well.

So, I think it should be possible, but you have to set up things yourself.

Hope this helps.
