To make a new databanks show up in the SOAP servers you have to add  
it to the mrs-config.xml file in /usr/local/etc/. If you edit this  
file while mrsws is running it will re-read the file each time it is  
saved.

It is possible to run mrsws without a config file, in that case it  
will read all databanks in the $MRS_DATA_DIR directory and then you  
have to 'kill -HUP' it to make it re-read the datafiles.
