Have a look at the attached script. This is one in the category, "you 
really shouldn't do this". But it works.

Basically, what it does is, it takes a SOAP envelope template and fills 
in the parameters. It then POST's this SOAP message to the address for 
the SOAP server and the reply is then printed. The reply is of course a 
SOAP message too. You can use standard XML parsing techniques to 
retrieve the data contained in it.

If you want to use this script to call blast services, you have to 
replace the namespace in the envelope with the one from the blast service.
