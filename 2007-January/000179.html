Hi,

I just checked in a first version for the mrsws-blast server. As the  
name implies, this is a SOAP based server that does blast searches in  
a MRS file. I don't have a good interface yet, but there is a test  
script in ws/blast called test.pl. You will have to edit this script  
to change the address from 'mini' to your local computer name.

Blast jobs are cached, if the exact same search comes by within an  
hour it is fetched from this cache. This is very useful for courses  
where students do the same search over and over.

The mrsws-blast application takes the same arguments as mrsws plus  
the '-t n' option which specifies how many threads to use to do the  
actual blast search.
