Hi,

I just checked in a new install script for MRS 3. Now if you have  
checked out mrs 3.0 you can simply run the install script to  
regenerate the plugin/make.config and ws/make.config files. The  
script will test the availability of all libraries needed to build  
both the Perl plugin and the mrsws web-services application. There is  
also a very simple new makefile in the root of this version that  
builds both the plugin and the mrsws application.

You can abort the script once it asks for the location of the data.  
At that point the make.config files have been written. This way you  
don't overwrite the scripts in your update_scripts and parser_scripts  
directories.
