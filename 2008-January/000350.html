- try running `mrs.pl info` on each of the parts to see if they are correct.

- Did the command you issued produce any error information?

- What is the exact version of MRS you're using? There was a bug in the
latest official release that might crash MRS when doing a merge on very
large files. The version rc1 (available at
ftp://ftp.cmbi.ru.nl/pub/software/mrs/mrs-3.1-rc1.tbz) fixes this
problem. You can also use the code from the subversion repository at
Berlios, you should use the trunk version.
