Op 14-sep-2007, om 21:20 heeft rbahena at ibt.unam.mx het volgende  
geschreven:

> Hi, I am trying to install MRS 3.0.1 on a linux machine, Red Hat  
> Enterprise 2.1,
> 64bit, Itanium processor.
>
> I installed gcc-4.1.1, swig-1.3.31, gsoap2.7.9d, boost_1_33_1 and  
> mrs script
> without problems.
>
> When I execute the next command:
>
>     mrs_query -d /data/mrs/ enzyme.cmp -q a
>
> I get the following error:
>
>    terminate called after throwing an instance of 'HError'
>       what():  Invalid mrs data file
>    Aborted

The -d option is the option for a databank, not for a directory. So  
you should enter the query:

mrs_query -d /data/mrs/enzyme.cmp -q a
                      ^^^ no space here

And if it still complains the same way, you might have a corrupt  
databank. To verify you might want to try the command:

mrs.pl info -d /data/mrs/enzyme.cmp

Hope this helps.
