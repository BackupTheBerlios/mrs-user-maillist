Maarten L. Hekkelman wrote:
> All,
> 
> I've decided to drop support for all OS'es except Debian derivatives 
> from MRS. The reason is that it is too much trouble to figure out all 
> the differences between operating systems.
> 
> The dependencies for MRS are:
> 
> - 64 bit OS. This means the development environment should be able to 
> build 64 bit executables and this appears to be less than trivial on 
> e.g. MacOS. (One of the main reasons MRS no longer works OK on MacOS).
> 
> - A recent boost version. And that is a challenge on more than one OS, a 
> lot of computers out there still come with boost 1.35 e.g.
> 
> - The version of the compiler is tricky too, I have come across quite a 
> few code generation bugs and I cannot guarantee I worked around all of them.
> 
> - GLIBC version problems.
> 
> And there are many more.
> 
> I will try to create a .deb package for installation using apt-get and 
> friends. This means that installation should be very easy if you have 
> Ubuntu or Debian and very difficult if you have anything else.
> 
> I'm sorry.
>
